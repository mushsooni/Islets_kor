# Islets 한글패치  


### 다운로드
https://github.com/mushsooni/Islets_kor/releases/download/v1.2.1/Islets_KR_1.2.1.zip

### 패치 방법
1. `Islets` 폴더 내부의 파일 5개를 게임 폴더에 붙여넣기  
(게임 폴더: 스팀 라이브러리 - 게임 우클릭 - 관리 - 로컬 파일 탐색)
2. (선택) `islets.exe_patcher.cmd` 실행
3. `Options` - `GAMEPLAY` - `LANGUAGE` - `한국어`로 변경
