# Islets 한글패치  


### 다운로드
https://github.com/mushsooni/Islets_kor/releases/download/v1.1/Islets_KR_1.1.zip

### 패치 방법
`Islets` 폴더 내부의 파일 3개를 게임 폴더에 붙여넣기  
(게임 폴더: 스팀 라이브러리 - 게임 우클릭 - 관리 - 로컬 파일 탐색)

### 알려진 문제
첫 번째 우편 발신자가 영어로 출력됨.  
자판기, 현수막 스프라이트 미번역.
